# Порядок запуска

## Запуск бэкенда

1. Установка пакетов
   `pipenv install`
2. Создание миграций
   `python orders/manage.py migrate`
3. Запуск сервера
   `python orders/manage.py runserver`

## Запуск фронтенда (React)

1. Переход в папку с форнтендом
   `cd orders/frontend`
2. Установка зависимостей
   `npm install`
3. Запуск
   `npm start`
